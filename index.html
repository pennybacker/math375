MATH/CS 375: Introduction to Numerical Computing (Fall 2016)

Prof. Matthew Pennybacker

Overview

This course is meant as an introduction to numerical analysis. We will study numerical methods used to solve linear and nonlinear equations, to interpolate and approximate data, and methods for numerical integration and differentiation. We will implement many of these algorithms in Matlab, and begin the course with a brief Matlab tutorial. Please see the syllabus for more information.
