MATH/CS 375: Introduction to Numerical Computing (Fall 2016)

Prof. Matthew Pennybacker

Overview

This course is meant as an introduction to numerical analysis. We will study numerical methods used to solve linear and nonlinear equations, to interpolate and approximate data, and methods for numerical integration and differentiation. We will implement many of these algorithms in Matlab, and begin the course with a brief Matlab tutorial. Please consult the syllabus for more information.

Samples

mvmultiply.m A function to perform matrix-vector multiplication (slowly).
mvtiming.m A script to measure the runtime of the mvmultiply function and plot the result.
bisect.m A function to perform the bisection method recursively.
newton.m A function to perform Newton-Raphson iteration.
backsub.m A function to solve an upper triangular system using back-substitution.
gausselim.m A function to perform Gaussian elimination without pivoting to obtain an equivalent upper triangular system.
